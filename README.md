# Rollerball

## How to Run:

### Using C++
1. make rollerball
2. ./scripts/run_servers.sh
3. ./scripts/run_ui.sh

### Using Python
1. make rollerball_py
2. ./scripts/run_servers.sh
3. ./scripts/run_ui.sh

Note: You will additionally need to install Pybind11. For Debian systems, run `sudo apt-get install pybind11-dev`.
